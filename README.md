# FreePBX Quick Provisioner

A FreePBX module for quick device provisioning.

## Installation

### Clone the Repository

You can clone this repository using SSH:

```bash
git clone git@github.com:Ezra90/freepbx-quickprovisioner.git
```

Or using HTTPS:

```bash
git clone https://github.com/Ezra90/freepbx-quickprovisioner.git
```

### FreePBX Module Installation

To install this module in FreePBX, clone it to the FreePBX modules directory:

```bash
cd /var/www/html/admin/modules
git clone git@github.com:Ezra90/freepbx-quickprovisioner.git quickprovisioner
```

The module should be installed at:
```
/var/www/html/admin/modules/quickprovisioner
```

### Enable the Module

After cloning the repository to the correct location, enable the module through the FreePBX admin panel:

1. Navigate to **Admin > Module Admin**
2. Find **Quick Provisioner** in the module list
3. Click **Enable** and then **Apply Configuration**

## Features

- Quick device provisioning for FreePBX
- Easy configuration and deployment
